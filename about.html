Sobre o QRCode Maker

O QRCode Maker é uma ferramenta simples e gratuita para gerar códigos QR para texto, links, Wi‑Fi, contatos, SMS, e-mails e telefones. O projeto foi criado por Vinícius Lima com o objetivo de oferecer uma solução leve e direta para criar QR codes sem custo.

Missão

Fornecer uma ferramenta online rápida e segura que permite a qualquer pessoa gerar QR codes para uso pessoal ou comercial, sem necessidade de cadastro.

Tecnologia

Este site é uma aplicação estática que utiliza HTML, CSS e JavaScript e é hospedada no Vercel. O código está disponível no GitHub.
